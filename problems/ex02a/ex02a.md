# ex02a

## 問題

１つ整数値を受け取った際に、偶数か奇数かを判定し、２回偶数を受け取るまで、無限while()ループさせるプログラムを作ってください。

## required function
```
- printf
- scanf
- if
- while(true)
```
### example terminal output
```
5
odd 0 times in while
6
odd 1 times in while
7
odd 1 times in while
8
odd 2 times, exit this program
```

### example terminal output 2
```
6
odd 1 times in while
8
odd 2 times, exit this program
```
